projects

Gameshow

A NodeJS based Gameshow which allows you to stream one client to a tv and controll the gameshow using your phone. The question sets will be stored in JSON files and can easily be customized.
The "master" can add questions to the gameboard, select them, start a timer, add players, assign them scores and more. All actions are displayed by the "display" client.

github.com/elYanuki/Gameshow

World Manager

A ElectronJS based desktop app ment to store a wiki and the corresponding map for fictional worlds.
You can create a world and upload a image to be used as the map, which you can add markers to. The markers come in 3 different sizes and can be moved arround and renamed, when clicked they link to a wiki entry. The wiki is based on the editorJS libary and allows you to create new pages and add elements like headings, tabels, lists and so on.

github.com/elYanuki/WorldManager
